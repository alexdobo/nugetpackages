Release-Notes for version 9.0.5222.0 (ASG-RD 2016):

Fixes
- Sorting of Favorites is working correctly
- Sync via csv file - if column 'Protocol' is empty, protocol will be set to 'RDP' as default
- Sync/Import csv file - column 'IP-Address' is set correctly
- If using one of the standard icons for connections these icons will also be used for tab/window image in connection control
- Screenshot function fixed for display scaling more than 100%

New features/enhancements
- Powershell API
- GeckoFX updated to version 45
- SQL-Express updated to version 2014
- Integrated new version of Putty (0.66)
- Integrated new RDP-ActiveX
- New global security options 'Move/copy items not allowed'
- Buttons in Navigation for 'Expand/Collapse all child items'
- New setting: Settings=>Connections=>Reconnect sessions in same window/tab
- New toolbar with following functionality
- Overview/Goto active connections
- Rotate active connections
- Layout-Manager: Save and Load layouts at any time
- New layout setting: Maximum text length of tab title
- Active Directory Sync: New option to sync computers of selected groups
- Active Directory Sync: New options for 'Mapping fields'
- Automatic save of offline data - new option in Settings=>Application; If turned on, option will be available at login process
- Quick-Search: New category in Settings dialog
- Quick-Search: New setting to extend search operations on description field also
- Quick-Search: New setting to use Regular Expression on search operations
- Added some new folder icons
- New setting 'Don't prompt when connecting multiple destinations'
- Added 'Send Ctrl-Alt-Del' for all RDP connections (also HyperV console session)
- Optional offline data synchronization (after reconnect to database)
